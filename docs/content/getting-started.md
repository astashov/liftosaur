## What is Liftosaur?

Liftosaur is a weightlifting tracker app that automates progressive overload. Pick a program (or create your own), follow the prescribed sets and reps, and the app automatically adjusts weights for your next workout based on your performance.

## Picking a Program

1. Open the app and tap **Choose Program**
2. Browse [the program library](/programs) or use filters to narrow down by experience level, frequency, or goal
3. Tap a program to see its details — exercises, sets/reps schemes, and progression rules
4. Tap **Add to Account** to start using it

## Starting a Workout

1. From the home screen, tap **Start Workout**
2. You'll see the exercises for today's session with prescribed sets, reps, and weights
3. Complete each set and tap the set circle to log it
4. If you complete all prescribed reps, the circle turns green. If not, it turns red
5. When you're done, tap **Finish Workout**

## Progressive Overload

After finishing a workout, Liftosaur automatically applies the program's progression rules. For example, if you completed all prescribed reps, the weight might increase by 5lb for next time. If you failed, the program might keep the same weight or apply a deload.

This is all handled by the program's built-in logic — you don't need to manually adjust anything.
