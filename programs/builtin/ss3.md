Great absolute beginner program from a famous trainer and powerlifter Mark Rippetoe. One of the most popular starting weightlifting routines.

```liftoscript
# Week 1
## Day A 1
Squat / 3x5 / 45lb / progress: lp(2.5lb, 1, 0, 10%, 2, 0)
Bench Press / 3x5 / 45lb / progress: lp(2.5lb, 1, 0, 10%, 2, 0)
Deadlift / 1x5 / 95lb / progress: lp(2.5lb, 1, 0, 10%, 2, 0)

## Day B 1
Squat / 3x5 / 45lb
Overhead Press / 3x5 / 45lb / progress: lp(2.5lb, 1, 0, 10%, 2, 0)
Chin Up / 3x10 / 0lb / progress: lp(2.5lb, 1, 0, 10%, 2, 0)

## Day A 2
Squat / 3x5 / 45lb
Overhead Press / 3x5 / 45lb
Power Clean / 5x3 / 95lb / progress: lp(2.5lb, 1, 0, 10%, 2, 0)

## Day B 2
Squat / 3x5 / 45lb
Bench Press / 3x5 / 45lb
Chin Up / 3x10 / 0lb
```
