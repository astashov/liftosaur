Good intermediate hypertrophy program. 4 day/week program, upper/lower body split.

```liftoscript
# Week 1
## Lower Body
Squat / 4x6-8 / 180s / progress: lp(5lb)
Stiff Leg Deadlift / 3x6-8 / 180s / progress: lp(5lb)
Leg Press / 3x10-12 / 120s / progress: lp(5lb)
Seated Leg Curl / 2x10-12 / 120s / progress: lp(5lb)
Standing Calf Raise / 3x6-8 / 180s / progress: lp(5lb)
Seated Calf Raise / 3x10-12 / 120s / progress: lp(5lb)

## Upper Body
Bench Press / 4x6-8 / 180s / progress: lp(5lb)
Bent Over Row / 3x6-8 / 180s / progress: lp(5lb)
Shoulder Press / 3x10-12 / 120s / progress: lp(5lb)
Chin Up / 2x10-12 / 120s / 0lb
Skullcrusher / 2x12-15 / 90s / progress: lp(5lb)
Hammer Curl / 2x12-15 / 90s / progress: lp(5lb)
```
