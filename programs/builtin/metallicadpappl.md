PPL program for beginners balancing "bro" muscles, but also giving enough intensity in the main movements

```liftoscript
# Week 1
## Monday - Pull
Deadlift / 1x5 / progress: lp(10lb, 1, 0, 10%, 1, 0)
Lat Pulldown / 3x10 / progress: lp(5lb, 1, 0, 10%, 1, 0)
Seated Row / 3x10 / progress: lp(5lb, 1, 0, 10%, 1, 0)
Face Pull, Cable / 5x20
Hammer Curl / 4x10
Bicep Curl / 4x10

## Tuesday - Push
Bench Press / 4x5, 1x5+ / progress: lp(5lb, 1, 0, 10%, 1, 0)
Overhead Press / 3x10 / progress: lp(5lb, 1, 0, 10%, 1, 0)
Incline Bench Press, Dumbbell / 3x10
Triceps Pushdown / 3x10
Triceps Extension / 3x10
Lateral Raise / 3x20

## Wednesday - Legs
Squat / 5x5 / progress: lp(5lb, 1, 0, 10%, 1, 0)
Romanian Deadlift, Barbell / 3x8 / progress: lp(5lb, 1, 0, 10%, 1, 0)
Leg Press / 3x10 / progress: lp(5lb, 1, 0, 10%, 1, 0)
Seated Leg Curl / 3x10 / progress: lp(5lb, 1, 0, 10%, 1, 0)
Standing Calf Raise / 5x10

## Friday - Pull
Bent Over Row / 4x5, 1x5+ / progress: lp(5lb, 1, 0, 10%, 1, 0)
Lat Pulldown / 3x10
Seated Row / 3x10
Face Pull, Cable / 5x20
Hammer Curl / 4x10
Bicep Curl / 4x10

## Saturday - Push
Overhead Press / 4x5, 1x5+ / progress: lp(5lb, 1, 0, 10%, 1, 0)
Bench Press / 3x10 / progress: lp(5lb, 1, 0, 10%, 1, 0)
Incline Bench Press, Dumbbell / 3x10
Triceps Pushdown / 3x10
Triceps Extension / 3x10
Lateral Raise / 3x20

## Sunday - Legs
Squat / 5x5
Romanian Deadlift, Barbell / 3x8
Leg Press / 3x10
Standing Calf Raise / 5x10
Seated Leg Curl / 3x10
```
