4 day a week program, first 2 days is "strength" - upper and lower, second 2 days is hypertrophy - upper and lower. 

```liftoscript
# Week 1
## Upper Power
Bench Press / 3x3-5 / 45lb / progress: lp(5lb)
Incline Bench Press, Dumbbell / 3x8-10 / 20lb / progress: lp(5lb)
Bent Over Row / 3x3-5 / 65lb / progress: lp(5lb)
Pull Up / 3x8-10 / 0lb / progress: lp(5lb)
Shrug, Barbell / 3x8-10 / 45lb / progress: lp(5lb)
Bicep Curl, Barbell / 3x8-10 / 45lb / progress: lp(5lb)
Skullcrusher / 3x3-5 / 30lb / progress: lp(5lb)

## Lower Power

Standing Calf Raise / 3x8-10 / 45lb / progress: lp(5lb)
Seated Calf Raise / 3x8-10 / 45lb / progress: lp(5lb)
```
